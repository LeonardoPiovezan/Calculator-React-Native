# Calculator-React-Native


### Summary

- [Requirements](#requirements)
- [Installation](#installation)

## Requirements

- Java SDK installed
- Node.js installed
- Xcode installed
- Android Studio installed
- npm
- React-Native


## Installation

Run this command in terminal:

```
npm install
```
And then:

For iOS build:
```
react-native run-ios
```

For Android version:

```
react-native run-android
```
